# Image upload

阅读 [Web Development with MongoDB and NodeJS](https://www.packtpub.com/product/web-development-with-mongodb-and-nodejs-second-edition/9781785287527) 搭建的一个简单的基于 MVC 模式图片上传应用。

使用：

- express
- multer
- art-template
- mongoose
- bulma
- jquery

只实现最基本的CRUD

## 使用

```shell
yarn

node server.js
```
